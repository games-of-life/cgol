# cgol

A [Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) in C.

All graphics done with [Raylib](https://www.raylib.com/)

## Installation

Install raylib.
Make is used to build.

## Usage

To compile into executable:

```bash
make
```

To run executable:

```bash
./bin/cgol
```
